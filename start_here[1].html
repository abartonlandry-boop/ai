Acerca do teu download

Podes encontrar as tuas informações nos ficheiros HTML deste download. Recomendamos que visualizes o download num computador para poderes utilizar estas ligações para navegar para as tuas informações. As ligações não funcionam em dispositivos móveis.

Num dispositivo móvel, usa as pastas nos downloads para aceder aos ficheiros HTML. Cada tipo de informação é armazenado no seu próprio ficheiro HTML e pode ser visualizada ao abrir o ficheiro.
